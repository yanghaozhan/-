# 环境的搭建

---

**Python环境**

介绍:scripts目录里面有解释器exe，和activate.ps1这两个文件

* 安装		路径终端输入   python -m venv   [你要的环境名称] 补充:这里的-m是因为没有激活环境时，指令会从系统环境变量里面找你当前路径，找不到D\Zhuomian
* 激活		终端输入一个·  按tab  出来D:\ZhuoMian\项目> .\.venv\Scripts\Activate.ps1，ps1是powershell的意思，左边显示一个绿色的表示激活成功。
* 导入		在这里pip install就是往这个环境里面下载必备的库。
* 退出		deactivate就是退出当前环境，这样的时候终端会启用你的c端默认Python。

---

**Jupyter环境= Python 环境 + 注册内核**

区别：jupyter环境是按照单元格去跑，变量保存在内存里，可以继续跑

* 安装环境		路径终端输入   python -m venv   [你要的环境名称]
* 激活环境		`.\jlab\Scripts\Activate.ps1`
* 装jupyterlab		`pip install jupyterlab` 可以浏览器打开(自带ipkernal)ipkernel是ipython kernel的缩写(kernel是内核的意思)
* 注册内核		`python -m ipykernel install --user --name jlab --display-name "你想要的名字"` 显示:Installed kernelspec jlab in C:\Users\27819\AppData\Roaming\jupyter\kernels\jlab
* 启动		`jupyter lab`
